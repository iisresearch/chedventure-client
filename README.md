Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## How do I play the demo game? ##
* Chedventure-server has to be running on localhost:4201 (also, run the migration with the database seeding).
* Go to localhost:4200/game.
* Click on the far right 'play' icon button in the table.
